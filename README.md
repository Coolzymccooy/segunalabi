# Segunalabi Portfolio

This repository contains the source code for **segunalabi.com**, a dark, premium portfolio showcasing the work of Alabi Oluwasegun Akeem. It’s built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and the app directory architecture.

## Running locally

To run this project locally you’ll need to have Node.js installed. Once you have Node installed you can install dependencies and run the development server:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Start the production build
npm start
```

## Overview

The project features:

* A home page with an animated hero, dynamic typewriter headlines and calls to action.
* A projects section listing featured and all projects with individual detail pages.
* A CV page where your CV can be viewed or downloaded.
* A contact page with direct contact details and an optional message form.

Feel free to customise the content and design to suit your needs.
